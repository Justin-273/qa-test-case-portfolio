# QA Test Case Portfolio

This repository contains sample QA test cases created using a mind mapping approach.

## Test Case Format
- Test Scenario
- Test Points
- Preconditions
- Test Steps / Test Data
- Expected Result

## Tools Used
- XMind (for test case design)
- Manual Testing Techniques
- GitHub (documentation & version control)

## Contents
- **test-cases/** – Manual test cases in mind map format
- **bug-reports/** – Sample bug reports
- **api-testing/** – Sample API test cases

## Purpose
This portfolio demonstrates my approach to efficient test case creation,
early test planning, and clear documentation in fast-paced development environments.
